# 异构CNN特征融合的多分支关注区域分析与正则化策略探索

## 项目简介

这是一个**人工智能课程期末实践项目**，探索了多分支卷积神经网络（CNN）的特征融合与调控策略。

项目使用 PyTorch 从零实现，适合深度学习初学者学习和参考。代码结构清晰，包含完整的模型定义、训练流程和可视化分析。

## 创新点

### 1. 异构分支设计

不同于传统单分支CNN，本项目设计了三种结构差异化的轻量CNN分支：

- **纹理分支**：浅层3×3卷积，捕获细粒度局部纹理
- **边缘分支**：中层5×5卷积，提取物体轮廓信息
- **语义分支**：深层7×7+3×3混合卷积，获取全局语义特征

三个分支的感受野依次增大，从不同尺度提取互补的特征表示。

### 2. 分支差异化与协作调控策略

提出了两种互补的正则化策略：

- **交叉协方差最小化约束**：通过最小化分支间特征向量的交叉协方差矩阵F范数，强制分支学习差异化的特征表示，减少信息冗余
- **协作奖励机制**：设计了一种"奖励而非惩罚"的融合策略——当融合分类性能超越最佳单分支时给予奖励，引导模型学习真正的互补信息

### 3. 基于Grad-CAM的分支关注区域量化分析

使用了一种量化多分支模型行为的直观方法：

- 使用Grad-CAM热力图可视化每个分支关注的图像区域
- 通过计算分支间热力图的IoU（交并比），量化分支关注区域的重叠程度
- 该指标可用于分析不同约束策略对分支行为的影响

## 部分实验结果（详情见/results/）

| 模型 | Test Accuracy | Mean IoU |
|------|--------------|----------|
| M0 单一大模型 | **90.42%** | - |
| M4 同构集成 | 87.57% | - |
| M5 异构零基线 | 87.48% | 0.1457 |
| M6 辅助监督最优 | **87.87%** | 0.1591 |
| M5 λ=0.01 去相关 | 86.82% | **0.0009** |

核心发现：
- 异构分支天然学习到差异化特征（无约束下IoU仅0.1457）
- 交叉协方差约束可将IoU降至接近零，但牺牲分类精度
- 辅助监督是当前架构下最有效的调控策略

## 项目结构

```
.
├── src/                    # 源代码
```

## 快速开始

### 环境要求

- Python 3.8+
- PyTorch 2.0+
- matplotlib
- numpy

### 运行示例

```bash
# 训练异构融合模型（仅辅助监督）
python src/main.py --model hetero_fusion_coop --lambda_orth 0 --lambda_coop 0 --lambda_aux 0.3 --epochs 80 --seed 42

# 训练去相关约束模型
python src/main.py --model hetero_fusion_coop --lambda_orth 0.01 --lambda_coop 0 --lambda_aux 0.3 --epochs 80 --seed 42

# 训练单一大模型（基线）
python src/main.py --model single_large --epochs 80 --seed 42
```
