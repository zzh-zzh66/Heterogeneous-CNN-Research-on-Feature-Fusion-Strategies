## 3. 各模型说明

### M1 — Single Branch1 (Texture / 纹理分支)
- **测试准确率**: 76.62%（最低）
- **参数量**: 0.057M（最小）
- **特点**: 浅层纹理特征提取，通道数最少（64维），适合边缘、纹理等低层特征
- **备注**: 模型最轻量，但性能有限，适合作为特征提取器组件

### M2 — Single Branch2 (Edge / 边缘分支)
- **测试准确率**: 85.33%
- **参数量**: 0.26M
- **特点**: 中层边缘/形状特征，通道数 64→128，适合捕捉中等复杂度模式
- **备注**: 性能接近 M3，参数量更少，性价比高

### M3 — Single Branch3 (Semantic / 语义分支)
- **测试准确率**: 85.32%
- **参数量**: 0.28M
- **特点**: 高层语义特征，通道数 128，适合目标级别识别
- **备注**: 与 M2 准确率几乎相同，但通道数更大

### M4 — Homo Ensemble (同构集成)
- **测试准确率**: 87.57%（最高）
- **参数量**: 0.80M（最大）
- **特点**: 3 个 Branch2 并行集成，拼接融合特征
- **备注**: 性能最佳，但计算成本最高（训练时间 42.4 min）

### M5 系列 — Hetero Fusion (异构融合)
- **最佳表现**: λ=0 时达到 87.48%
- **特点**: 3 个异构分支（纹理+边缘+语义）拼接融合，加入正交正则化
- **λ 正则化影响**:
  - λ=0（无正则）: 87.48%
  - λ=0.01: 86.82%（↓0.66%）
  - λ=0.05: 85.95%（↓1.53%）
  - λ=0.1: 85.43%（↓2.05%）
  - λ=0.5: 85.93%（↓1.55%）
- **备注**: λ 值越大，正则化越强，特征解耦越明显，但性能有所下降。λ=0 时异构融合与同构集成效果接近。

---

## 4. 关键发现

1. **异构 vs 同构**: λ=0 时异构融合（87.48%）接近同构集成（87.57%），但参数量更少（0.63M vs 0.80M）

2. **λ 正则化作用**: λ 从 0 增到 0.1 时，准确率下降约 2%，说明过度约束会损害性能

3. **三分支分工**: Branch1(76%) / Branch2(85%) / Branch3(85%) 准确率差异显著，符合低/中/高层特征提取的理论预期

4. **最佳性价比**: Hetero Fusion λ=0 以 0.63M 参数达到 87.48%，优于同构集成的 0.80M
