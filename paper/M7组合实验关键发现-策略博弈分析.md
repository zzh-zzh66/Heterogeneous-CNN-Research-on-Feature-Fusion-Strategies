# M7组合实验关键发现：特征层去相关与决策层辅助及合作的三角博弈

## 实验背景

M5纯差异化约束实验发现 λ=0.01 可将分支间 Mean IoU 压至 0.0009，但分类准确率从 87.48% 降至 86.82%。这引出一个核心问题：**决策层的辅助损失（λa）和合作损失（λc）能否补偿特征层去相关（λ）带来的准确率损失？**

为此设计 M7 组合实验，首次同时启用交叉协方差约束（λ=0.01）、辅助损失（λa=0.3）与合作奖励（λc=0/0.05）。

***

## 全部 M7 实验结果

| 模型                    |     λ    |    λc    |    λa   |  Test Acc  |  Mean IoU  | 实验性质       |
| --------------------- | :------: | :------: | :-----: | :--------: | :--------: | ---------- |
| M5\_λ0（零基线）           |     0    |     -    |    -    |   87.48%   |   0.1457   | 无任何约束      |
| M5\_λ0.01（纯去相关）       |   0.01   |     -    |    -    | **86.82%** | **0.0009** | 仅特征层去相关    |
| **M7\_#1（去相关+辅助）**    | **0.01** |   **0**  | **0.3** | **86.58%** | **0.2698** | 特征层 + 决策辅助 |
| **M7\_#2（去相关+辅助+合作）** | **0.01** | **0.05** | **0.3** | **85.72%** | **0.2773** | 三个损失全部启用   |

***

## 发现一：损失叠加链逐层瓦解去相关效果

```
纯去相关 (M5_λ0.01):       IoU = 0.0009   ← 几乎完美差异化
+ 辅助损失 (M7_#1):         IoU = 0.2698   ← 辅助强制分支看判别区
+ 合作奖励 (M7_#2):         IoU = 0.2773   ← 合作让分支更趋同
```

三个损失形成了一条单向拉力链。辅助损失强制每个分支独立分类 → 分支必须定位最具判别力的图像区域 → 判别区域天然共享 → IoU 回升。合作损失在此基础上奖励"分支预测一致" → 进一步推动分支关注趋同 → IoU 继续上升。

**辅助损失几乎完全抵消了去相关效果（IoU 从 0.0009 暴增至 0.2698），合作损失强化了这一趋势。**

***

## 发现二：合作损失让准确率不升反降

| 对比                 |          Acc 变化          |          IoU 变化         |
| ------------------ | :----------------------: | :---------------------: |
| M5\_λ0.01 → M7\_#1 | 86.82% → 86.58% (-0.24%) | 0.0009 → 0.2698 (+300×) |
| M7\_#1 → M7\_#2    | 86.58% → 85.72% (-0.86%) | 0.2698 → 0.2773 (+2.8%) |

合作损失不仅未能挽救准确率，反而使其进一步恶化。低于 M5\_λ0.01 的 86.82%，更低于 M5\_λ0.001 的 85.67%。**M7\_#2 是差异化约束组中 Acc 最低的模型。**

原因分析：

1. 合作奖励机制是"融合是否优于最优单分支"的比较。当 λ=0.01 强制各分支产出低质量特征时，三个分支的分类水平都在下降，"融合优于单分支"的奖励信号丧失参考价值。
2. Train Acc 93.59% vs Test Acc 85.72%（差距 7.9%），表明多个损失联合作用使 loss landscape 更复杂，过拟合加剧。

***

## 发现三：三角博弈的最优解不是全开

| 策略组合                    |     Acc    |   IoU  | 综合评价                 |
| ----------------------- | :--------: | :----: | -------------------- |
| 纯去相关 (λ 仅)              |   86.82%   | 0.0009 | 差异化最优，但牺牲 Acc        |
| 去相关 + 辅助 (λ+λa)         | **86.58%** | 0.2698 | **兼顾 Acc 与 IoU 的折中** |
| 去相关 + 辅助 + 合作 (λ+λa+λc) |   85.72%   | 0.2773 | Acc 最差，IoU 最高        |

**最优组合点位于 λ+λa（M7\_#1），而非三者全开。** 三个策略并非单调叠加就更好——它们之间存在非线性的权衡关系。

***

## 发现四：判别性区域天然共享——"完全去相关"与"有效分类"存在根本矛盾

交叉协方差最小化只保证特征向量线性无关，不保证无关的特征对分类有用。辅助损失在 M7\_#1 中揭示了：一旦要求分支用去相关的特征做分类，它们被迫回到判别性区域，IoU 随即反弹。

> **差异化程度的提高存在一个上限，超过这个上限，差异化就变成"降质"——降低特征判别力而非增加互补信息。**
